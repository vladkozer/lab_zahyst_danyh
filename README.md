# Шифрування зображень методом блочної перестановки та LFSR

## Опис
Даний репозиторій містить реалізацію алгоритму шифрування зображень,
заснованого на розбитті зображення на блоки та їх перестановці
за псевдовипадковою послідовністю, згенерованою за допомогою
лінійного регістра зсуву зі зворотним зв’язком (LFSR).

Алгоритм реалізовано відповідно до наукової публікації 2021 року.
Для демонстрації використовується одне статичне зображення,
яке імітує кадр відеопотоку.

## Алгоритм роботи
1. Зчитування вхідного зображення
2. Розбиття зображення на блоки фіксованого розміру
3. Генерація псевдовипадкової послідовності за допомогою LFSR
4. Перестановка блоків відповідно до згенерованого порядку
5. Формування зашифрованого зображення
6. Розшифрування шляхом зворотної перестановки (для аналізу)
